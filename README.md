# 🚀 Nuxt Minimal Starter

Project ini adalah aplikasi berbasis **Nuxt 3 + Tailwind CSS** yang telah dideploy ke Vercel.

🌐 **Live Demo:**
https://cmlabs-frontend-fulltime-test-sandy.vercel.app/

---

## 📦 Tech Stack

* Nuxt 3
* Tailwind CSS
* Pinia
* Vercel (Deployment)

---

## ⚙️ Setup Project

Pastikan kamu sudah menginstall:

* Node.js (disarankan v18+)
* Package manager (npm / pnpm / yarn / bun)

---

## 📥 Install Dependencies

Jalankan salah satu perintah berikut:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

---

## 💻 Menjalankan di Local

Untuk menjalankan project di local:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

Buka di browser:

```
http://localhost:3000
```

---

## 🏗️ Build Production

Untuk build project:

```bash
# npm
npm run build

# pnpm
pnpm build

# yarn
yarn build

# bun
bun run build
```

---

## 👀 Preview Production

Untuk melihat hasil build secara lokal:

```bash
# npm
npm run preview

# pnpm
pnpm preview

# yarn
yarn preview

# bun
bun run preview
```
